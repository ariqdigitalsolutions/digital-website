How this static website currently uses local browser storage and third-party links.

1. What this website currently uses

The website stores a small local preference in your browser after you accept or dismiss the cookie notice. This prevents the same notice from appearing on every visit.

2. Essential storage

This preference does not create a user profile and is not used for advertising. The website's core pages can still be read if local browser storage is unavailable, although the notice may reappear.

3. Third-party links

Opening WhatsApp, social media, maps or other external services may allow those providers to set their own cookies or collect usage information. Their policies apply once you leave this website.

4. Future analytics or marketing tools

No analytics or advertising scripts should be added without first reviewing consent requirements and updating this notice. Where consent is legally required, non-essential tools should remain disabled until the visitor chooses to allow them.

5. Managing browser data

You can remove site data through your browser settings. Doing so will reset the saved cookie-notice preference.
